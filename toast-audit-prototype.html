Как делают топ RPG
Genshin Impact
Pickup: маленький floating text у подобранного предмета.
Equip: никакого тоста — предмет просто появляется в слоте с flash-анимацией.
Ошибки: красный текст по центру экрана (1 сек).
Diablo Immortal
Loot: текст плавает над предметом. Legendary = экранная вспышка.
Equip: звук + визуальное обновление слота.
Тосты: только ошибки и системные (потеря связи).
AFK Arena
Rewards: отдельный экран-церемония.
Equip: inline анимация + звук.
Тосты: почти нет. Всё через inline UI.
Clash Royale
Rewards: popup/modal с анимацией открытия.
Действия: кнопка меняет состояние (disabled/success).
Тосты: не используют вообще.

Inline feedback на элементах
Ошибки внизу, церемонии отдельно
3-уровневая система нотификаций
СЛОЙ 1 Inline Feedback — на самом элементе
Equip/Unequip: flash-рамка на слоте + haptic → убрать тост
Sell: "+120" floating text + монеты к счётчику → убрать тост
Heal: HP bar green flash + "+50 HP" floating → убрать тост
Repair: слот меняет border с красного на нормальный → убрать тост
Upgrade: level badge пульсирует + "+5" → убрать тост
Stats/Stance: кнопка → "Saved ✓" state → убрать тост
Claim: кнопка → "Claimed ✓" disabled → убрать тост
Social: кнопка → "Sent ✓" disabled → убрать тост
СЛОЙ 2 Bottom Toast — только ошибки и срочное
Ошибки: "Not enough gold", "Connection failed", "Fight failed" → bottom + Retry/Shop action
Broke: "Sword broke!" → bottom + "Repair" action button
Limits: "Daily limit reached" → bottom, 3 sec
Max 1 toast, дедупликация, swipe-down dismiss
СЛОЙ 3 Celebration Banner — milestone события
Level Up: баннер под навбаром + звук + вибрация
Achievement: баннер с названием + описанием
Rank Up: баннер с новым рангом
Rare+ Drop: баннер для Epic/Legendary лута
Quest Complete: баннер
Dungeon Clear: баннер
Отдельная система от тостов, никогда не перекрывает навигацию
