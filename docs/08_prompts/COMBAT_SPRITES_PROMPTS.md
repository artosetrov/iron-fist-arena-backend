## ИНСТРУКЦИЯ ПО ГЕНЕРАЦИИ

1. **Формат:** Всегда выбирай 1:1 (квадрат)
2. **Копируй промпт целиком** — каждый промпт самодостаточный, содержит все стилевые указания
3. **Генерируй по одному** — один промпт = один спрайт
4. **Начинай с Приоритета 1** — это минимум для рабочего боя
5. **Проверяй:** спрайт должен быть на чистом чёрном фоне с толстыми ink outlines для лёгкого удаления фона
6. **Если результат плохой** — перегенерируй тот же промпт, Nano Banana Pro даёт разные результаты каждый раз
