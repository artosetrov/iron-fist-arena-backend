# Hexbound — Промпты для Nano Banana Pro: Спрайты боссов

> **Status**: `canonical` — Active prompt collection. Indexed at `docs/08_prompts/ASSET_PROMPTS_INDEX.md`

## Общие правила для ВСЕХ спрайтов боссов

**Формат:** 1:1 (квадрат)
**Фон:** Warm cream/beige parchment background
**Стиль:** Hand-drawn ink and watercolor illustration, pen-and-ink linework with watercolor wash coloring, grotesque caricature with exaggerated proportions, dark fantasy cartoon style similar to classic tabletop RPG book illustrations. Rich vibrant color palette — bold saturated greens, oranges, purples, reds. Visible brushstrokes and ink splatters. Expressive asymmetrical eyes, drool, warts, bumps, scars, grime and dirty details. Characters are ugly-funny — repulsive yet humorous. Thick hand-drawn outlines with varying line weight.
**Ракурс портретов:** Head and upper shoulders, cropped at chest level, slightly angled 3/4 view
**Ракурс полных фигур:** Full body visible, dynamic pose, same art style
**Важно:** Каждый босс имеет ДВА промпта — портрет (бюст) и полная фигура (боевая поза)

# ИНСТРУКЦИЯ ПО ГЕНЕРАЦИИ

1. **Формат:** Всегда выбирай 1:1 (квадрат)
2. **Фон:** Кремовый/бежевый (warm cream/parchment) — НЕ чёрный и НЕ белый!
3. **Копируй промпт целиком** — каждый промпт самодостаточный
4. **Генерируй по одному** — один промпт = одна картинка
5. **Порядок приоритетов:**
   - Сначала портреты финальных боссов (Arena Warden, Lich King Verath, Pyrox the Eternal)
   - Затем все портреты по порядку данжей
   - Затем полные фигуры в том же порядке
6. **Проверяй стиль:** должен совпадать с существующими портретами героев:
   - Гротескный cartoon с акварельной раскраской
   - Thick ink outlines с variable line weight
   - Яркие насыщенные цвета — bold saturated palette
   - Преувеличенные пропорции — большие головы, выразительные лица
   - Грязные детали — бородавки, слюни, шрамы, грязь
   - Уродливо-забавные — repulsive yet humorous
7. **Если результат не в стиле — добавь к промпту:**
   - "pen-and-ink linework with watercolor wash coloring"
   - "grotesque caricature, ugly-funny character"
   - "rich vibrant color palette, visible brushstrokes"
   - "similar to old school tabletop RPG book illustrations"
8. **Если результат плохой** — перегенерируй тот же промпт
