# Hexbound — Промпты для Nano Banana Pro: Гербы классов (экран выбора класса)

> **Status**: `canonical` — Active prompt collection. Indexed at `docs/08_prompts/ASSET_PROMPTS_INDEX.md`

## Общие правила для ВСЕХ гербов классов

**Формат:** 1:1 (квадрат), 4K
**Фон:** Pure solid black (#000000) — однородный, для лёгкого вырезания
**Стиль:** Hand-drawn ink and watercolor illustration, pen-and-ink linework with watercolor wash coloring, crosshatching shading technique, grotesque dark fantasy medieval manuscript art style. Thick bold outlines with varying line weight.
**Композиция:** Геральдическая — центральный элемент (щит/медальон/печать класса) на переднем плане, за ним скрещённое оружие класса. Компактная иконка-герб, как эмблема гильдии или рыцарского ордена. Все элементы плотно собраны в единую композицию, ничего не разбросано.
**Важно:** БЕЗ персонажей, БЕЗ текста. Чистые края, толстые аутлайны — иконка должна легко вырезаться с фона. Никаких элементов, выходящих за рамки композиции.
