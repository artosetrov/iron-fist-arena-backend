## Паттерны и уроки

### 1. Каскадные фиксы challenges route (8 коммитов на 1 файл)

**Что произошло:** `backend/src/app/api/social/challenges/route.ts` получил 7 последовательных fix-коммитов после начального feat. Каждый исправлял одну ошибку: missing await, wrong arg count, wrong field name, wrong enum value.

**Корневая причина:** Агент не прочитал source файлы game-lib функций перед вызовом. Правило "Before using any function — check its signature" в CLAUDE.md уже существовало, но было нарушено.

**Воздействие:** 7 итераций push-check-fix вместо одной. ~30 минут потерянного времени.

**Урок:** Правило существует, но не помогло. Нужна **проактивная проверка Oracle** при создании новых route файлов. Oracle scanner (`check_async_await.sh`) проверяет только missing await, но не проверяет:
- Количество аргументов функций
- Имена полей возвращаемых типов (`.stamina` vs `.current`)
- Корректность enum values

**Действие:** Добавить в Oracle SKILL.md чек-лист для новых route файлов.

### 2. CodingKeys — молчаливый decode failure

**Что произошло:** 6+ моделей (SocialStatus, FriendsListResponse, IncomingChallenge и др.) имели explicit CodingKeys с snake_case маппингом, но backend отправлял camelCase. Decode молча возвращал nil/defaults.

**Корневая причина:** `.convertFromSnakeCase` в APIClient + explicit CodingKeys = decoder ищет literal snake_case в JSON, но находит camelCase.

**Воздействие:** Guild Hall badge никогда не показывался. Challenges не загружались. Обнаружено только при ручном тестировании.

**Урок:** Правило добавлено в CLAUDE.md (`72dc0fb`). Scanner не проверяет CodingKeys-паттерн.

**Действие:** Можно добавить grep-проверку в Guardian scanner — найти Swift модели с explicit CodingKeys содержащими `_` (snake_case), когда APIClient использует `.convertFromSnakeCase`.

### 3. [String: Any] не конформит Encodable

**Что произошло:** `MessageService.swift` использовал `[String: Any]` body в `APIClient.post()` — generic typed метод ожидает `Encodable`.

**Корневая причина:** Привычка из JS/TS мира, где dictionary — это стандартный тип для JSON body.

**Действие:** Правило уже в CLAUDE.md. Guardian scanner может добавить grep для `[String: Any]` в аргументах `post()`.
