## Паттерны и уроки

### 1. ButtonStyle + @Environment — новый паттерн `EnabledReader`

**Что:** `@Environment(\.isEnabled)` нельзя напрямую использовать в `ButtonStyle`, потому что `makeBody()` — не `View`, а функция. Прямое использование вызывает runtime warning "Accessing Environment<Bool>'s value outside of being installed on a View".

**Фикс:** Создан private struct `EnabledReader<Content: View>` который читает `@Environment(\.isEnabled)` в контексте View и передаёт значение через closure.

**Применено к:** PrimaryButtonStyle, CompactPrimaryButtonStyle, DangerCompactButtonStyle, NeutralButtonStyle (4 стиля).

**Статус:** Паттерн уже зафиксирован в коде. Стоит добавить правило в CLAUDE.md для будущих стилей.
