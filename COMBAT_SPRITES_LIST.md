# Hexbound — Список боевых спрайтов / VFX

## Боевая система (сводка)

- **4 типа урона:** physical, magical, poison, true_damage
- **4 класса:** Warrior (physical), Tank (physical), Rogue (poison), Mage (magical)
- **Исходы удара:** попадание, крит (×1.5), уворот (dodge, до 30%), промах
- **Оружие:** мечи, кинжалы, булавы, молоты, посохи, жезлы, скипетры
- **Зоны атаки/защиты:** голова, торс, ноги
- **Особые эффекты предметов:** огненный урон, яд при ударе, вампиризм, воскрешение

## СТИЛЬ СПРАЙТОВ

Все спрайты должны быть:
- **На прозрачном/чёрном фоне** (для наложения)
- **В стиле игры** — ink crosshatching, hand-drawn, гротескный dark fantasy
- **Квадратные** (1:1) или с прозрачными краями
- **Яркие и читаемые** на тёмном фоне боя
- **С толстыми ink outlines** для консистентности со стилем персонажей

### Цветовая кодировка:
- Physical → золотисто-оранжевый (#D4A537, #FFD700)
- Magical → синий-фиолетовый (#4169E1, #8B5CF6)
- Poison → ядовито-зелёный (#2ECC71, #50C878)
- True Damage → белый-золотой (#FFFFFF, #FFD700)
- Fire → красно-оранжевый (#E63946, #FF6B35)
- Heal → зелёный-золотой (#2ECC71, #D4A537)
- Buff → голубой (#4FC3F7)
